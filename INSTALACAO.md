## ✅ OPÇÃO 1: USAR LOCALMENTE (SEM INTERNET)

### No Computador:
1. ✅ Clique 2x em `index.html`
2. ✅ Salve nos favoritos
3. ✅ Pronto! Funciona offline

### No Celular:
**Você precisa COPIAR os arquivos para o celular**

#### Via AirDrop (iPhone):
1. Selecione os 3 arquivos:
   - `index.html`
   - `manifest.json`
   - `service-worker.js`
2. Clique com botão direito → Compartilhar → AirDrop
3. Envie para seu iPhone
4. Abra `index.html` no Safari
5. Compartilhar → Adicionar à Tela Inicial

#### Via Google Drive/Dropbox:
1. Faça upload dos 3 arquivos
2. Baixe no celular
3. Abra `index.html` no navegador
4. Instale como app

**⚠️ PROBLEMA:** Celular pode ter dificuldade de abrir arquivo local

## 🚀 COMO COLOCAR ONLINE (5 MINUTOS)

### Método 1: GitHub Pages (Mais Simples)

1. **Criar conta no GitHub** (grátis)
   - Vá em https://github.com
   - Clique em "Sign up"

2. **Criar repositório**
   - Clique no "+" → "New repository"
   - Nome: `projeto-60-dias`
   - Marque "Public"
   - Clique "Create repository"

3. **Fazer upload dos arquivos**
   - Clique em "uploading an existing file"
   - Arraste os 3 arquivos:
     - `index.html`
     - `manifest.json`
     - `service-worker.js`
   - Clique "Commit changes"

4. **Ativar GitHub Pages**
   - Vá em "Settings" (engrenagem)
   - Clique em "Pages" no menu lateral
   - Em "Source": selecione "main"
   - Clique "Save"
   - **Aguarde 2 minutos**

5. **SEU LINK ESTARÁ PRONTO:**
   ```
   https://JoaoBueno1.github.io/projeto-60-dias/
   ```

6. **Abra no celular e instale!**

---

### Método 2: Netlify (Mais Rápido)

1. **Acesse** https://app.netlify.com/drop
2. **Arraste a pasta** `dieta` completa
3. **Pronto!** Link gerado na hora
4. Exemplo: `https://projeto-60-dias-xyz.netlify.app`

---

### Método 3: Vercel (Profissional)

1. **Acesse** https://vercel.com
2. **Clique** "Add New..." → "Project"
3. **Faça upload** da pasta
4. **Deploy automático**
5. Link: `https://projeto-60-dias.vercel.app`
