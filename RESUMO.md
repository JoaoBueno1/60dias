## 🎯 TREINOS DA SEMANA

| Dia | Treino | Tempo |
|-----|--------|-------|
| **Seg** | Peito + Tríceps | 60 min + cardio |
| **Ter** | Costas + Bíceps | 60 min + cardio |
| **Qua** | ⚽ Futebol + Ombros | 90 + 30 min |
| **Qui** | Core ou Descanso | 30-45 min |
| **Sex** | Pernas + Tríceps | 60 min + cardio |
| **Sáb** | Costas + Bíceps + Antebraço | 60 min + cardio |
| **Dom** | ⚽ Futebol + Ombros | 90 + 30 min |

### Frequência Semanal:
- **Peito:** 2x
- **Costas:** 2x
- **Pernas:** 1x + 2x futebol
- **Ombros:** 2x (Qua + Dom)
- **Bíceps:** 2x
- **Tríceps:** 2x
- **Cardio:** 5x (incluindo futebol)

## 🚀 COMO USAR

### 1️⃣ Instalação

**Opção A: Local (sem internet)**
- Abrir `index.html` no navegador
- Salvar nos favoritos

**Opção B: Online (RECOMENDADO)**
- Hospedar no GitHub Pages
- Instalar como PWA
- Usar em qualquer lugar

### 2️⃣ Uso Diário
1. Fazer check-in diário pela manhã
2. Clicar nos cards de treino para ver exercícios
3. Marcar dieta/treino/cardio

### 3️⃣ Uso Semanal
1. Registrar medidas toda segunda (jejum)
2. Ver gráficos de evolução
3. Ler insights automáticos

### 4️⃣ Configuração
1. Definir metas (peso, gordura, cintura)
2. Acompanhar progresso
3. Exportar backup mensal
